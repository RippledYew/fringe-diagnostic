# Fringe Diagnostic

A terminal-based system health monitoring tool built for the Fringe Lab.

## Features
-Cpu usage, frequency, and temperature monitoing
-Ram and swap tracking
-Disk usage with snap mount filterin
-Network latency check
-Fringe Toolkit status panel
-Top processes by memory
-Intel/NVIDIA GPU panel
-Uptime display
-System status pass/fail summary

## Usage
'''bash
diagnostic
'''

## Requirements
- Python 3
- psutil
- rich
